# 🔐 Phishing Website Detection

## 📌 Project Overview
This project aims to detect phishing websites using Machine Learning techniques. 
It classifies a website as **Legitimate (0)** or **Phishing (1)** based on various features.

## 🎯 Objective
To build a model that can identify whether a website is safe or malicious.
